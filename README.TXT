PROJECT TITLE: Word-Fighter
PURPOSE OF PROJECT: To create a fun word game in a team environment, implement effective data structures, and have fun!
VERSION or DATE: 1.0.0, November 9th, 2021
HOW TO START THIS PROJECT: Click the "run" button to start the project.
AUTHORS: Eric Zhang, Kevin Wang
USER INSTRUCTIONS: 
Click the letters in the letterbox to form words. 
Click the green thumbs-up button once you have formed a word. 
Make words to attack the enemy, and survive as long as you can!

Data structures used:
ArrayList<String> in WordFetcher.java
GreenfootImage[][] in Animation.java
ArrayList<Letter> in FinishedWordBox.java
char[] consonants and char[] vowels in Letter.java
Letters[][] letters and char[] arr in LetterBox.java
